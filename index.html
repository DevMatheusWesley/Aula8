Pseudoelemento é uma palavra-chave usada para estilizar uma parte específica de um determinado elemento.

Por diversas vezes, você está em uma página da internet e encontra elementos da página, como links, que mudam de cor no momento que passamos o mouse sobre eles. Isso pode ser feito utilizando pseudoclasses no CSS. Pseudoclasse é uma palavra-chave usada para aplicar um efeito especial a um seletor, ou seja, ela define um estado especial para um determinado elemento HTML. A pseudoclasse seleciona elementos para aplicar uma determinada ação, propriedade etc.
